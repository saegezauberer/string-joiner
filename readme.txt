Please create Spring Boot application that will have one controller with one method. Method should call two services in parallel. First service should have a single method that returns String as a response with a possibility to set the value of the beforementioned String at the runtime. Second service should also have a single method that should return String as a response. Value of the String for the second service should be the name of the current day. Controller method should combine the responses from the services and return them to the caller.
